# SN7413

## Summary
SN7413 is a dual 4-input NAND Schmitt-trigger gate.

## Functional Intent
Implement two independent logical 4-input NAND channels.

## Interface
- `input logic [1:0] a`
- `input logic [1:0] b`
- `input logic [1:0] c`
- `input logic [1:0] d`
- `output logic [1:0] y`

## Behavioral Description
For each channel `i`: `y[i] = ~(a[i] & b[i] & c[i] & d[i])`.

## Truth Table / State Table
Per channel: output is low only when all four inputs are high.

## Modeling Assumptions
- Schmitt-trigger hysteresis is abstracted away in this digital logic model.
- Logical NAND behavior only.

## Abstractions and Non-Goals
- No analog threshold/hysteresis modeling.
- No propagation delay or electrical behavior modeling.

## Verification
- Exhaustive packed-vector simulation over all `a`, `b`, `c`, and `d` values.
