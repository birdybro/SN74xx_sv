# SN7427

## Summary
SN7427 is a triple 3-input NOR gate.

## Functional Intent
Implement three independent logical 3-input NOR channels.

## Interface
- `input logic [2:0] a`
- `input logic [2:0] b`
- `input logic [2:0] c`
- `output logic [2:0] y`

## Behavioral Description
For each channel `i`: `y[i] = ~(a[i] | b[i] | c[i])`.

## Truth Table / State Table
Per channel: output is high only when all three inputs are low.

## Modeling Assumptions
- Ideal logical model only.

## Abstractions and Non-Goals
- No propagation delay or electrical behavior modeling.

## Verification
- Exhaustive packed-vector simulation over all `a`, `b`, and `c` values.
