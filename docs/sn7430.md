# SN7430

## Summary
SN7430 is an 8-input NAND gate.

## Functional Intent
Implement one logical 8-input NAND function.

## Interface
- `input logic [7:0] a`
- `output logic y`

## Behavioral Description
`y = ~(&a)`.

## Truth Table / State Table
Output is low only when all eight inputs are high.

## Modeling Assumptions
- Ideal logical model only.

## Abstractions and Non-Goals
- No propagation delay or electrical behavior modeling.

## Verification
