# SN74138

## Summary
SN74138 is a 3-to-8 decoder with active-low outputs and three enable inputs (`G1`, `G2A_n`, `G2B_n`).

## Functional Intent
Decode a 3-bit select into one asserted output line when enabled.

## Interface
- `input logic g1` (active-high enable)
- `input logic g2a_n` (active-low enable)
- `input logic g2b_n` (active-low enable)
- `input logic [2:0] sel`
- `output logic [7:0] y_n` (active-low outputs)

## Behavioral Description
- When enabled (`g1 && !g2a_n && !g2b_n`):
  - `y_n[sel] = 0`
  - all other `y_n` bits are `1`
- When disabled:
  - all `y_n` bits are `1`

## Truth Table / State Table
Mode summary:

| g1 | g2a_n | g2b_n | y_n |
|---|---|---|---|
| 1 | 0 | 0 | one-hot active-low decode of `sel` |
| otherwise | x | x | all ones |

## Modeling Assumptions
- `sel[2:0]` is interpreted as an unsigned index `0..7` into `y_n`.
- Logical model only (no hazards or timing-skew effects).

## Abstractions and Non-Goals
- No propagation delay or electrical fanout modeling.

## Verification
- Exhaustive simulation over all 64 combinations of enables and `sel`.
- Checks disabled mode all-ones and enabled mode one-hot active-low decode.
