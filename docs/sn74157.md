# SN74157

## Summary
SN74157 is a quad 2-to-1 multiplexer with common select and active-low enable.

## Functional Intent
Select between two 4-bit input groups with one shared select control.

## Interface
- `input logic g_n` (active-low enable)
- `input logic sel` (shared select)
- `input logic [3:0] a`
- `input logic [3:0] b`
- `output logic [3:0] y`

## Behavioral Description
- If `g_n=1`, all outputs are forced low (`y=4'b0000`).
- If `g_n=0`, output is selected per bit:
  - `sel=0` -> `y=a`
  - `sel=1` -> `y=b`

## Truth Table / State Table
Mode summary:

| g_n | sel | y |
|---|---|---|
| 1 | x | 0000 |
| 0 | 0 | a |
| 0 | 1 | b |

## Modeling Assumptions
- Disabled output state is modeled as logical zero.
