# SN7417

## Summary
SN7417 is a hex non-inverting buffer/driver with open-collector outputs.

## Functional Intent
Provide six independent non-inverting buffer channels while documenting open-collector abstraction boundaries.

## Interface
- `input logic [5:0] a`
- `output logic [5:0] y`

## Behavioral Description
For each channel `i`: `y[i] = a[i]`.

## Modeling Assumptions
- Open-collector behavior is abstracted to logical buffering with an implicit pull-up.
- Wired bus interaction across multiple drivers is out of scope.

## Abstractions and Non-Goals
- No analog/electrical drive modeling.
- No propagation delay modeling.

## Verification
- Exhaustive simulation over all packed `a` input combinations.
