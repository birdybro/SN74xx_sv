# SN7404

## Summary
SN7404 is a hex inverter. This model provides six independent logical inversion channels in one synthesizable module.

## Functional Intent
Implement logical inversion for each channel.

## Interface
- `input  logic [5:0] a`: inverter inputs
- `output logic [5:0] y`: inverted outputs

## Behavioral Description
For each channel `i` in `[0:5]`:
- `y[i] = ~a[i]`

## Truth Table / State Table
Per channel:

| a | y |
|---|---|
| 0 | 1 |
| 1 | 0 |

## Modeling Assumptions
- Ideal logical inversion only.
- No timing, loading, or power-pin modeling.

## Abstractions and Non-Goals
- No propagation delay modeling.
- No electrical characteristics or package behavior.

## Verification
- Exhaustive simulation over all 64 `a` combinations.
- Expected output computed as `~a`.
