# SN7426

## Summary
SN7426 is a quad 2-input NAND gate/buffer with open-collector outputs.

## Functional Intent
Provide four independent NAND channels while documenting open-collector abstraction boundaries.

## Interface
- `input logic [3:0] a`
- `input logic [3:0] b`
- `output logic [3:0] y`

## Behavioral Description
For each channel `i`: `y[i] = ~(a[i] & b[i])`.

## Modeling Assumptions
- Open-collector behavior is abstracted to logical NAND output with an implicit pull-up.
- Wired bus interaction across multiple drivers is out of scope.

## Abstractions and Non-Goals
- No analog/electrical drive modeling.
- No propagation delay modeling.

## Verification
- Exhaustive simulation over all packed `a` and `b` input combinations.
