# SN7416

## Summary
SN7416 is a hex inverting buffer/driver with open-collector outputs.

## Functional Intent
Provide six independent inverting buffer channels while documenting open-collector abstraction boundaries.

## Interface
- `input logic [5:0] a`
- `output logic [5:0] y`

## Behavioral Description
For each channel `i`: `y[i] = ~a[i]`.

## Modeling Assumptions
- Open-collector behavior is abstracted to logical inversion with an implicit pull-up.
- Wired bus interaction across multiple drivers is out of scope.

## Abstractions and Non-Goals
- No analog/electrical drive modeling.
- No propagation delay modeling.

## Verification
- Exhaustive simulation over all packed `a` input combinations.
