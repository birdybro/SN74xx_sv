# SN7405

## Summary
SN7405 is a hex inverter with open-collector outputs.

## Functional Intent
Provide six independent inverter channels while documenting open-collector abstraction boundaries.

## Interface
- `input logic [5:0] a`
- `output logic [5:0] y`

## Behavioral Description
For each channel `i`: `y[i] = ~a[i]`.

## Truth Table / State Table
Per channel:

| a | y |
|---|---|
| 0 | 1 |
| 1 | 0 |

## Modeling Assumptions
- Open-collector behavior is abstracted to logical inversion with an implicit pull-up.
- Wired-AND/wired-OR interaction between multiple drivers is out of scope for this single-device model.

## Abstractions and Non-Goals
- No analog/electrical current-drive modeling.
- No bus contention or resistor network modeling.
- No propagation delay modeling.

## Verification
- Exhaustive simulation over all packed `a` input combinations.
