# SN74153

## Summary
SN74153 is a dual 4-to-1 multiplexer with common select lines and separate active-low enables.

## Functional Intent
Select one of four data inputs per channel based on shared select lines.

## Interface
- `input logic [1:0] g_n` (active-low channel enables)
- `input logic [1:0] sel` (shared select)
- `input logic [1:0] i0, i1, i2, i3` (per-channel data inputs)
- `output logic [1:0] y`

## Behavioral Description
For each channel `ch`:
- If `g_n[ch]` is `1`, output is forced low (`y[ch]=0`).
- If `g_n[ch]` is `0`, output follows selected input:
  - `sel=00 -> i0[ch]`
  - `sel=01 -> i1[ch]`
  - `sel=10 -> i2[ch]`
  - `sel=11 -> i3[ch]`

## Truth Table / State Table
Mode summary per channel:

| g_n | sel | y |
|---|---|---|
| 1 | xx | 0 |

| 0 | 01 | i1 |
| 0 | 10 | i2 |
| 0 | 11 | i3 |

## Modeling Assumptions
- Disabled channel output is modeled as logic `0`.
- Shared select applies to both channels.
