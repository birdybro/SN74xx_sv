# SN7414

## Summary
SN7414 is a hex Schmitt-trigger inverter.

## Functional Intent
Implement six independent inverting channels.

## Interface
- `input logic [5:0] a`
- `output logic [5:0] y`

## Behavioral Description
For each channel `i`: `y[i] = ~a[i]`.

## Truth Table / State Table
Per channel:

| a | y |
|---|---|
| 0 | 1 |
| 1 | 0 |

## Modeling Assumptions
- Schmitt-trigger hysteresis is abstracted away in this digital logic model.
- Logical inversion behavior only.

## Abstractions and Non-Goals
- No analog threshold/hysteresis modeling.
- No propagation delay or electrical behavior modeling.

## Verification
- Exhaustive packed-vector simulation over all `a` values.
