# SN74138 Work Record

- Chip name: SN74138
- Category: decoders
- Logical function: 3-to-8 decoder/demultiplexer with active-low outputs
- Expected ports: g1, g2a_n, g2b_n, sel[2:0], y_n[7:0]
- Expected behavior:
  - if enabled (`g1=1`, `g2a_n=0`, `g2b_n=0`): one selected output low, others high
  - if disabled: all outputs high
- Verification plan: exhaustive control/select combinations
- Known ambiguities: output ordering vs select bits; modeled as `sel` numeric index
- Implementation status: complete
